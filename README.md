## The result that surprised me

I went in expecting RoBERTa — trained specifically on Twitter data — to win.

It didn't.

A zero-shot Gemini 2.0 Flash, with no task-specific training, outperformed every fine-tuned transformer I tested across 10,000+ real tweets.

| Model | F1 | Type |
|---|---|---|
| 🥇 Gemini 2.0 Flash | **0.800** | Zero-shot LLM |
| 🥈 LLaMA 3.1 8B | 0.768 | Zero-shot LLM |
| distilBERT | 0.680 | Fine-tuned |
| RoBERTa (Twitter) | 0.670 | Fine-tuned |
| VADER | 0.640 | Rule-based |

**Takeaway:** Domain alignment matters less than scale and instruction-following ability — at least at this task complexity.
