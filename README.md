## 🔬 Featured Project — LLM Sentiment Benchmarking Study

> *Does domain-specific fine-tuning beat zero-shot LLMs? I tested it empirically.*

Designed and ran a systematic benchmark of **5 NLP models** (VADER → distilBERT → RoBERTa → LLaMA 3.1 8B → Gemini 2.0 Flash) against 10,000+ real tweets from the Sentiment140 dataset.

**Key finding:** Zero-shot LLMs outperformed fine-tuned transformers. Gemini 2.0 Flash achieved **F1: 0.80**, beating domain-trained RoBERTa (F1: 0.67) — proving that scale and instruction-following ability compensate for lack of fine-tuning.

| Model | F1 Score | Type |
|---|---|---|
| Gemini 2.0 Flash | **0.800** ✅ | Zero-shot LLM |
| LLaMA 3.1 8B | 0.768 | Zero-shot LLM |
| distilBERT | 0.680 | Fine-tuned transformer |
| RoBERTa | 0.670 | Fine-tuned transformer |
| VADER | 0.640 | Rule-based baseline |

**Stack:** Python · Groq API · OpenRouter · HuggingFace · pandas · Matplotlib
